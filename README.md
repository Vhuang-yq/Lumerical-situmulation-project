基于 Lumerical 的马赫-曾德尔干涉仪（Mach-Zehnder Interferometer）光学仿真项目。

项目简介：
本项目使用 Lumerical FDTD Solutions 对马赫-曾德尔干涉仪进行仿真，研究其干涉特性、传输谱以及结构参数对器件性能的影响。MZI 是集成光学中最基础且最重要的器件之一，广泛应用于光开关、光调制器、波分复用器等领域。

仿真设置:
软件版本: Lumerical 2024 
求解器: FDTD Solutions
主要参数:
工作波长: 1550 nm
波导材料: 硅
波导宽度: 0.5 um
波导高度: 0.22 um
